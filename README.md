# kerimuenver.de

Portfolio website.

## Tech Stack

Next.js, TypeScript, Tailwind CSS, Motion

## Development

```bash
npm install
npm run dev
```

## Deployment

Static export, deployed to [Cloudflare Pages](https://pages.cloudflare.com/).

Contact form uses [Web3Forms](https://web3forms.com/).
